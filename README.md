# 🌌 星轨 STARL — 你的人生意义坐标

> Soul Trace And Resonance of Life

24道题 · 6个维度 · 16种人生意义原型

融合星座元素、依恋理论与存在主义哲学，找到属于你的那颗星，和它背后的人生意义。

## 特性

- 🔥 6个核心维度：星焰/潮汐/锚/探索/月镜/光
- 🌟 16种人生意义原型
- 🎨 暗色宇宙主题 + 星空动画
- 📱 完全响应式，手机/平板/PC适配
- ⚡ 纯前端，零依赖，极速加载
- 🔒 无需后端，不采集任何用户数据

## 部署

本项目是纯静态单页应用，支持任何静态托管：

- **GitHub Pages**: Fork 后在 Settings → Pages 中开启
- **Vercel**: 导入仓库即可自动部署
- **Netlify**: 拖拽文件夹即可部署
